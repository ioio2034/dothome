CSS Reference

CSS

CSS(Cascading Style Sheets)은 HTML을 꾸며주는 스타일 시트입니다.

  • HTML은 웹 사이트의 몸체를 담당한다면, CSS는 옷과 악세사리 같이 꾸미는 역할을 담당합니다.

CSS Color

CSS 색상 표현은 4가지 방법이 있습니다.

  • RGB/RGBA
    • RGB(Red, Green, Blue) : 각각의 변수에는 0~255사이의 정수를 입력하여 색상을 표현할 수 있다.
    • RGBA(Red, Green, Blue, Alpha) : RGB 표현의 확장으로, Alpha(불투명도) 변수의 값은 0.0(투명) ~ 1.0(불투명) 사이의 값 중 지정한다.
  • HSL/HSLA
    • HSL(Hue, Saturation, Lightness) : 색상, 채도, 명도로 0~360까지의 색상환에 따라 색상을 표현할 수 있다. 0:red, 120:green, 240:blure로 표현한다.
    • HSLA(Hue, Saturation, Lightness, Alpha) : HSL 표현의 확장으로, Alpha(불투명도) 변수의 값은 0.0(투명) ~ 1.0(불투명) 사이의 값 중 지정한다.
  • 헥사코드
    • #RRGGBB 형식의 16진수 색상 값을 이용하여 색상을 표현할 수 있다.
    • RR, GG, BB는 각각 Red, Green, Blue를 나타내는 값이며 16진수 00에서 FF사이의 값으로 결정된다.
  • 키워드
    • 색상의 이름(키워드)으로도 색상을 표현할 수 있다.
    • Color 키워드표

CSS 선언 방법

CSS 선언하는 방법은 3가지 있습니다.

  • 외부 스타일 시트(External Style Sheet) : 외부 CSS 파일과 연결
  • 내부 스타일 시트(Internal Style Sheet) : Head 태그 안에 정의
  • 인라인 스타일 시트(Inline Syyle Sheet) : 해당 태그 안에 정의

float으로 인한 영역 깨짐(height:0) 방지법

float을 사용하면 height 값을 0으로 인식합니다.

  1. 깨지는 영역에 똑같이 float을 사용합니다. (사용하지 X)
  2. float의 성질을 차단하는 clear:both를 사용합니다. (복잡한 레이아웃의 경우 깨진 영역을 찾기 어려움)
  3. float을 사용한 부모박스한테 overflow: hidden을 사용합니다. (많이 사용 O)
  4. float을 사용한 부모태그한테 clearfix를 사용합니다.
.clearfix {*zoom:1;} /* ie7 핵*/
.clearfix:before, .clearfix:after {display: table; content: ''; line-height: 0;}
.clearfix:after {clear:both;}

이미지 표현 방법

웹 문서에서 이미지를 표현 할 수 있는 방법은 3가지가 있습니다.
  • <img> 태그를 통해서 넣는 방법 : 이미지에 의미가 있을 때
  • CSS background를 통해서 넣는 방법 : 이미지가 의미가 없을 때, 장식
  • 이미지 스프라이트 기법을 통해서 넣는 방법

이미지 스프라이트

이미지 스프라이트(image sprite)란 여러 개의 이미지를 하나의 이미지로 합쳐서 한번에 다운받는 시스템입니다.
  • 여러번의 서버 요청을 한번으로 줄일 수 있습니다.
  • 이미지 수정이나 관리가 간편합니다.
  • 단점:웹 접근성을 준수하기 위해서는 IR기법를 설정해야 합니다.

IR 기법

IR 기법은 이미지 대체텍스트 제공을 위한 CSS기법 입니다.

스크린리더 사용자를 위한 IR(Image Replacement)기법

  • Text-indent를 활용한 IR기법
    • 속성 값에 현재 사용되고 있는 디스플레이의 해상도보다 작은 음의 정수 값을 주는 것입니다.
    • 예를 들면 span{text-indent: -9999}와 같이 사용할 수 있습니다.
    • 단점
      : 사용자의 단말기에 따라 이미지가 제대로 로드되지 않을 때 이미지를 설명하는 텍스트를 보고 콘텐츠의 내용을 확인할 수 없다.
      : text-indent 스타일 속성이 적용된 요소가 많을 때 컴퓨터가 웹페이지 로드 시 위치값을 그만큼 많이 계산해 하므로 성능에 저하를 불러올 수 있다.
  • CSS의 위치 속성과 overflow: hidden을 사용하는 방법
    • 텍스트가 있는 <span> 또는 <div> 등의 요소에 디스플레이 해상도 보다 훨씬 작은 음의 정수 값으로 위치를 부여한 다음, 높이와 너비를 1px 정도로 작게 설정합니다.
    • overflow: hidden과 position: absolute로 대체 텍스트를 화면에서 완전히 숨깁니다.
  • 높이와 넓이를 0으로 설정하는 IR기법
    • 대체 텍스트 글자가 들어가는 등의 요소의 높이와 너비를 0으로 지정하는 방법입니다.
    • 대체할 텍스트 요소에 BackGround 이미지를 설정한 다음 대체 텍스트가 포함되는 요소의 높이와 너비를 0으로 지정합니다.
  • z-index를 활용한 IR기법
    • 이미지의 대체 텍스트에 z-index 속성 값을 음의 정수로 부여하는 방식입니다.
    • 브라우저에서 CSS를 끄거나, 어떠한 이유로 웹 페이지에 적용된 CSS가 정상적으로 로드되지 않을 때 숨겨진 텍스트가 화면에 출력됩니다.
    • position 속성을 사용하기 때문에 디바이스의 사양에 따라 성능에 영향을 줄 수 있습니다.
  • + 참고
    • Bootstrap 프레임워크를 사용하여 웹페이지를 개발할 경우 sr-only라는 클래스가 정의되어 있습니다.
    • IR 기법을 사용할 때 이 클래스를 유용하게 사용할 수 있습니다.

컨텐츠를 보이지 않게 하는 방법

  1. display : none <=> display : block // 영역 사라짐 // 애니메이션X
  2. visibility : hidden <=> visibility : visble // 영역 유지 // 애니메이션X
  3. opacity : 0 <=> opacity : 1 // 영역 유지 // 애니메이션O
  4. width : 0; height : 0; overflow:hidden(IR효과)

position:relative를 사용해야 하는 경우(시험)

  • position:aboslute의 기준점이 필요할 때
  • z-index를 사용해야 하는 경우 포지션 값이 없으면 reletive를 사용하여 z-index의 값을 인식시킬수 있습니다.
  • 요소의 위치를 유지한 채 left, top, right, bottom의 위치 값을 사용하고 싶을 때

컨텐츠 요소를 가운데 오게 하는 방법

  1. 가장 기본적인 방법, 한 줄(요소)일 경우에서만 해당이 됩니다.
    • 부모요소(인라인구조) - text-align : center; line-height: (height값과 동일하게)
    • 요소(블록구조) - margin:0 auto; line-height: (height값과 동일하게)
  2. 요소의 성질을 table로 변환시켜서 가운데 정렬을 하는 방법, table자체를 레이아웃에 많이 사용하지 않기 때문에 잘 쓰지 않는 방법입니다.
    • 부모요소 - display:table;
    • 자식요소 - display:table-cell, text-align:center; vertical-align:middle;
  3. 절대 요소를 이용한 가운데 정렬방법, 제일 흔하게 쓰이지만 단점은 영역이 없어지는 것이기 때문에 반응형을 할 때 사용하기 불편한 점이 있습니다.
    • 부모요소 - position:relative;
    • 자식요소 - position:absolute; top:50%; left: 50%; margin-top:-(자식 요소 높이값의 반); margin-left:-(자식 요소 가로값의 반);
  4. 만약 자식요소의 가로값과 세로 값을 모르고, 유동적인 경우에 절대 요소를 사용하여 가운데로 오는 정렬방법입니다.
    • 부모요소 - position:relative;
    • 자식요소 - position:absolute; top:50%; left: 50%; transform:translate(-50%, -50%);
  5. 절대 요소를 이용한 가운데 정렬 방법, 흔하게 쓰이지 않습니다.
    • 부모요소 - position:relative;
    • 자식요소 - position:absolute; top:0; left:0; top:0; right:0; margin: auto;
  6. CSS3를 이용한 방법, 최신 브라우저 이외에서는 작동하지 않는 단점이 있지만 표준화 된다면 앞으로 많이 쓰일 방법입니다.
    • 부모요소 - height 값 설정
    • 자식요소 - display: flex; justify-content: center; align-items: center;

Font Awesome 자동접근성

  • 아이콘을 폰트화하여 제공해주는 대표적인 것 중 Font Awesome은 백그라운드 이미지를 굳이 쓰지 않더라도 여러 아이콘을 클래스를 입력하는 것만으로 폰트로 표현이 가능합니다.
  • 아이콘 폰트 클래스를 적용할 경우 의미 없는 문자 코드가 css 콘텐츠에 포함되어 있어 스크린리더가 이러한 무의미한 텍스트를 읽게 될 수 있습니다.
  • aria-hidden=”true” 를 사용하여 이러한 의미 없는 요소들이 음성출력되지 않도록 해 주어야 합니다.
    예: <i class="fa fa-fighter-jet" aria-hidden="true"></i>
  • 의미를 가진 아이콘이라면 아이콘에 대한 의미를 스크린리더 사용자에게도 전달해 주어야 합니다.
  • 의미 있는 아이콘이라 하더라도 일반 텍스트에 아이콘에 대한 내용을 포함하는 경우에는 아이콘에 대한 설명을 생략가능
  • 상황에 따라 aria-label 혹은 css를 활용한 스크린리더 텍스트를 추가, 스크린리더 사용자를 위한 추가 설명 제공하기 article을 참고하여 추가 정보 제공이 가능합니다.
  • 아이콘에 대한 설명을 툴팁 형태로 title 속성을 추가할수 있으나 해당 요소는 대부분의 스크린리더에서 지원되지 않습니다.
  • font awesome cdn을 사용하고 자동 접근성을 설정하면 의미 있는 아이콘에 title 속성을 추가하면 자동으로 span 태그를 생성하고 화면에서는 숨기고 스크린리더만 인식 가능한 스크린리더 텍스트를 추가해주게 됩니다.
    예: <dt><i title="Time to destination by car" class="fa fa-car" aria-hidden="true"></i><span class="sr-only">Time to destination by car</span></dt>
  • Font Awesome 로그인하여 환경설정 페이지로 이동해서 Font Awesome cdn 방식을 스크립트 방식으로 설정해야 자동 접근성 기능을 사용할 수 있습니다.

CSS Description

width

요소의 가로 값을 정의합니다.

height

요소의 세로 값을 정의합니다.


background

요소의 배경을 정의합니다.

  • background-attachment : 배경 이미지의 고정 여부를 설정합니다.
  • background-blend-mode : 배경을 혼합했을 때의 상태를 설정합니다.
  • background-clip : 배경을 혼합했을 때의 상태를 설정합니다.
  • background-color : 백그라운드의 색상을 설정합니다.
  • background-image : 백그라운드를 설정합니다.
  • background-origin : 백그라운드 영역 속성을 설정합니다.
  • background-position : 백그라운드 위치를 설정합니다.
  • background-repeat : 백그라운드 반복 여부를 설정합니다.
  • background-size : 백그라운드 사이즈를 설정합니다.

background-color

백그라운드 색상을 정의합니다.

  • background-color: #fff - 백그라운드 색을 설정합니다.
  • background-color: transparent - 백그라운드 색을 투명하게 설정합니다.

background-image

백그라운드 이미지 속성을 정의합니다.

  • background-image: none - 백그라운드 이미지 속성을 설정하지 않습니다.
  • background-image: url("이미지 경로") - 백그라운드 이미지를 경로로 설정합니다.

background-position

백그라운드 이미지 위치를 설정합니다.

  • background-position : left top
  • background-position : left center
  • background-position : left bottom
  • background-position : right top
  • background-position : right center
  • background-position : right bottom
  • background-position : center top
  • background-position : center center
  • background-position : center bottom
  • background-position : 10px 40px
  • background-position : 10% 5%

background-repeat

백그라운드 이미지의 반복 여부를 설정합니다.

  • background-repeat : repeat(기본값) - 이미지를 X축 Y축으로 반복 설정합니다.
  • background-repeat : repeat-x = 이미지를 X축으로 반복 설정합니다.
  • background-repeat : repeat-y = 이미지를 Y축으로 반복 설정합니다.
  • background-repeat : no-repeat = 이미지를 반복하지 않고, 하나만 나오게 합니다.

text-align

텍스트의 정렬 방식을 정의합니다.

  • text-align : left - 텍스트의 정렬을 왼쪽으로 설정합니다.
  • text-align : right - 텍스트의 정렬을 오른쪽으로 설정합니다.
  • text-align : center - 텍스트의 정렬을 가운데로 설정합니다.
  • text-align : justify - 텍스트의 정렬을 양쪽으로 설정합니다.

text-transform

텍스트를 대문자 또는 소문자로 바꾸는 속성입니다.

  • text-transform : none - 속성을 적용하지 않습니다. 입력된 그대로 출력합니다.
  • text-transform : capitalize - 각 단어의 첫번째 글자만 대문자로 변경합니다.
  • text-transform : uppercase - 모든 텍스트를 대문자로 바꿉니다.
  • text-transform : lowercase - 모든 텍스트를 소문자로 바꿉니다.

margin

박스 요소의 바깥쪽 여백을 설정합니다.

  • margin-top: 10px - 요소의 위쪽 바깥 여백을 설정합니다.
  • margin-right: 11px - 요소의 오른쪽 바깥 여백을 설정합니다.
  • margin-bottom: 12px - 요소의 아래쪽 바깥 여백을 설정합니다.
  • margin-left: 13px - 요소의 왼쪽 바깥 여백을 설정합니다.
  • margin : 10px 11px 12px 13px - 위쪽/오른쪽/아래쪽/왼쪽 바깥 여백
  • margin : 10px 11px 12px - 위쪽 10px/왼쪽,오른쪽 11px/아래쪽 12px 바깥 여백
  • margin : 10px 11px - 위쪽,아래쪽 10px/왼쪽,오른쪽 11px 바깥 여백
  • margin : 10px - 위쪽,오른쪽,아래쪽,왼쪽 10px 바깥 여백
  • margin : auto - 기본값, 자동값
  • margin : 0, auto - 블록구조를 가운데 정렬

padding

박스 요소의 안쪽 여백을 설정합니다.

  • padding-top: 10px - 요소의 위쪽 안쪽 여백을 설정합니다.
  • padding-right: 11px - 요소의 오른쪽 안쪽 여백을 설정합니다.
  • padding-bottom: 12px - 요소의 아래쪽 안쪽 여백을 설정합니다.
  • padding-left: 13px - 요소의 왼쪽 안쪽 여백을 설정합니다.
  • padding : 10px 11px 12px 13px - 위쪽/오른쪽/아래쪽/왼쪽 안쪽 여백
  • padding : 10px 11px 12px - 위쪽 10px/왼쪽,오른쪽 11px/아래쪽 12px 안쪽 여백
  • padding : 10px 11px - 위쪽,아래쪽 10px/왼쪽,오른쪽 11px 안쪽 여백
  • padding : 10px - 위쪽,오른쪽,아래쪽,왼쪽 10px 안쪽 여백

font-size

폰트 크기 속성을 설정합니다.

  • font-size : xx-small - 폰트의 크기를 xx-small로 설정합니다.
  • font-size : x-small - 폰트의 크기를 x-small로 설정합니다.
  • font-size : small - 폰트의 크기를 small로 설정합니다.
  • font-size : medium - 폰트의 크기를 medium로 설정합니다. 기본값:16px
  • font-size : large - 폰트의 크기를 large로 설정합니다.
  • font-size : x-large - 폰트의 크기를 x-large로 설정합니다.
  • font-size : xx-large - 폰트의 크기를 xx-large로 설정합니다.
  • font-size : smaller - 폰트의 크기를 부모 요소보다 작은 크기로 설정합니다.
  • font-size : larger - 폰트의 크기를 부모 요소보다 큰 크기로 설정합니다.
  • font-size : length 14px - 폰트의 크기를 px,cm 등으로 크기를 설정합니다.
  • font-size : 20% - 폰트의 크기를 부모(상위) 요소의 글꼴 크기에 대한 백분율로 설정합니다.

line-height

문장과 문장 사이의 간격을 설정합니다.

  • line-height : normal - 웹브라우저에서 정한 기본값으로 설정합니다.
  • line-height : number ex) 3 - 폰트 크기의 몇 배 크기로 할지 지정합니다.
  • line-height : length ex) 18px - 길이로 줄 높이를 지정합니다.
  • line-height : percentage ex) 20% - 폰트 크기를 기준으로 백분율로 줄 높이를 지정합니다.
  • line-height = height : 문장이 한 줄일때 중간으로 오는 효과

position

요소의 위치를 설정합니다.
  • position : static(기본값)
  • position : absolute - 요소의 위치를 절대값으로 설정합니다.
  • position : relative
    1. position : absolute의 기준점이 필요할 때 사용합니다.
    2. 요소의 위치를 유지한 채로 left, top, right, bottom의 위치 값을 사용하고 싶을 때
    3. z-index를 사용하고 싶은데 포지션이 없으면 relative을 사용하여 z-index값을 인식 시킬 수 있습니다.
  • position : fixed - 요소의 위치를 고정으로 설정합니다.

border

테두리 속성을 설정합니다.
  • border-width: 테두리 두께 속성을 설정합니다.
  • border-style: 테두리 스타일 속성을 설정합니다.
  • border-color: 테두리 색 속성을 설정합니다.
  • border-image: 테두리 이미지 속성을 설정합니다.
  • border-radius: 테두리 굴곡을 설정합니다.

border-width

테두리 두께 속성을 설정합니다.
  • border-top-width : 요소의 위쪽 테두리의 너비를 설정합니다.
  • border-right-width : 요소의 오른쪽 테두리의 너비를 설정합니다.
  • border-bottom-width : 요소의 아래쪽 테두리의 너비를 설정합니다.
  • border-left-width : 요소의 왼쪽 테두리의 너비를 설정합니다.
  • border-width : 10px 11px 12px 13px - 위쪽/오른쪽/아래쪽/왼쪽 테두리 너비
  • border-width : 10px 11px 12px - 위쪽/오른쪽,왼쪽/아래쪽 테두리 너비
  • border-width : 10px 11px - 위쪽,아래쪽/오른쪽,왼쪽 테두리 너비
  • border-width : 10px - 모든 테두리 너비
  • border-width : thin(1px) - 얇은 테두리를 지정합니다.
  • border-width : medium(3px) - 기본값, 중간 테두리를 지정합니다.
  • border-width : thick(5px) - 두꺼운 테두리를 지정합니다.
  • border-width : length - 테두리의 두께를 정의할 수 있습니다.
  • border-width : inherit - 부모 요소에서 이 속성을 상속받습니다.

border-style

테두리 스타일 속성을 설정합니다.
  • border-top-style : 요소의 위쪽 테두리를 설정합니다.
  • border-right-style : 요소의 오른쪽 테두리를 설정합니다.
  • border-bottom-style : 요소의 아래쪽 테두리를 설정합니다.
  • border-left-style : 요소의 왼쪽 테두리를 설정합니다.
  • border-style : none - 기본값, 테두리가 없습니다.
  • border-style : hidden - 테두리를 숨깁니다.
  • border-style : dotted - 점선 테두리를 정의합니다.
  • border-style : dashed - dotted보다는 굵은 점선 테두리를 정의합니다.
  • border-style : solid - 단색 테두리를 정의합니다.
  • border-style : double - 이중 경계를 정의합니다.
  • border-style : groove - 3D 그루브 경계를 정의 합니다. 효과는 테두리 색 값에 따라 다릅니다.
  • border-style : ridge - 3D 능선 테두리를 정의합니다. 효과는 테두리 색 값에 따라 다릅니다.
  • border-style : inset - 3D 삽입 테두리를 정의합니다. 효과는 테두리 색 값에 따라 다릅니다.
  • border-style : outset - 3D 시작 테두리를 정의합니다. 효과는 테두리 색 값에 따라 다릅니다.

border-color

테두리 색 속성을 설정합니다.
  • border-color : color코드 - 테두리의 색을 지정합니다.
    • color코드는 #000000, RGB, RGBA, HSL, HSLA로 쓸 수 있습니다.
  • border-color : red green blue pink - 위쪽/오른쪽/아래쪽/왼쪽 테두리 색
  • border-color : red green blue - 위쪽/오른쪽,왼쪽/아래쪽 테두리 색
  • border-color : red green - 위쪽,아래쪽/오른쪽,왼쪽 테두리 색
  • border-color : red - 모든 테두리 색
  • border-color : transparent - 테두리를 투명하게 지정합니다.

border-image

list-style

목록 스타일 속성을 설정합니다.

letter-spacing

글자 사이의 간격을 지정합니다.
  • letter-spacing : normal - 기본적인 간격을 지정합니다.
  • letter-spacing : length - 글자 사이의 간격을 직접 지정하며 음수값도 허용합니다.
  • letter-spacing : inherit - 속성 값을 부모 요소로부터 상속받습니다.

word-spacing

단어 사이의 간격을 지정합니다.
  • word-spacing : normal - 기본적인 간격을 지정합니다.
  • word-spacing : length - 단어 사이의 간격을 직접 지정하며 음수값도 허용합니다.
  • word-spacing : inherit - 속성 값을 부모 요소로부터 상속받습니다.

overflow

요소의 박스가 해당 콘텐츠를 넘쳤을 때 어떤 식으로 표현할 지를 결정할 수 있는 속성입니다.
  • overflow : visible - 기본값, 영역에서 벗어난 부분을 자르지 않고 모두 보여줍니다.
  • overflow : hidden - 영역에서 벗어난 부분을 숨겨서 보이지 않게 지정합니다.
  • overflow : scroll - 영역에서 벗어난 부분을 숨기되 스크롤을 생성하여 볼 수 있게 지정합니다.
  • overflow : auto - 영역을 벗어난 부분이 있으면 자동으로 스크롤을 생성하게 지정합니다.
  • overflow : inherit - 속성 값을 부모 요소로부터 상속받습니다.

cursor

요소에 마우스 포인터가 위치할 경우 커서의 타입(모양)을 지정합니다.
  • cursor : auto - 기본값, 브라우저에서 기본적으로 지원하는 모양으로 지정합니다.
  • cursor : crosshair - 십자모양의 커서
  • cursor : default - 기본 흰색 화살표 모양의 커서
  • cursor : help - 기본 화살표 모양 옆에 물음표 아이콘 모양의 커서
  • cursor : move - 브라우저 상에서 창 이동을 나타내는 십자모양 화살표 커서
  • cursor : pointer - 손모양의 커서
  • cursor : progress - 기본 화살표 모양 옆에 로딩 중일 때 나타나는 모래시계 아이콘 모양의 커서
  • cursor : text - 텍스트에 마우스 가져다 대면 나타나는 일자 모양의 커서
  • cursor : wait - 로딩 중일 때 나타나는 모래시계 모양의 커서
  • cursor : e-resize - 좌우 검정 화살표 모양의 커서, 동쪽으로 사이즈 조절
  • cursor : n-resize - 상하 검정 화살표 모양의 커서, 북쪽으로 사이즈 조절
  • cursor : ne-resize - 북동쪽 검정 화살표 모양의 커서,북동쪽으로 사이즈 조절
  • cursor : nw-resize - 북서쪽 검정 화살표 모양의 커서,북서쪽으로 사이즈 조절
  • cursor : s-resize - 상하 검정 화살표 모양의 커서,남쪽으로 사이즈 조절
  • cursor : se-resize - 남동쪽 검정 화살표 모양의 커서,남동쪽으로 사이즈 조절
  • cursor : sw-resize - 남서쪽 검정 화살표 모양의 커서,남서쪽으로 사이즈 조절
  • cursor : w-resize - 좌우 검정 화살표 모양의 커서,서쪽으로 사이즈 조절

text-overflow

글자들이 속한 영역에서 벗어났을 때 어떤 방식으로 보여줄지를 지정합니다.
  • text-overflow : clip - 문자를 해당 영역까지만 나타내고 자릅니다.
  • text-overflow : ellipsis - 문자를 해당 영역까지만 표현하고 “…”을 사용하여 표시합니다.
  • text-overflow : string - 지정된 문자열을 출력합니다.

white-space

요소안에서 자동 줄 바꿈 여부를 나타내는 값을 설정합니다.
  • white-space : normal - 기본값, 연속공백과 줄바꿈을 통합하여 처리합니다.
  • white-space : nowrap - 연속공백은 통합하고 줄바꿈은 하지 않습니다.
  • white-space : pre - 연속공백, 줄바꿈 모두 하지 않고 원본 그대로 유지됩니다.
  • white-space : pre-line - 연속 공백은 메꾸어져 하나의 공백이 됩니다. 행의 줄 바꿈은 행의 박스를 메꾸기 위해 필요한 때에 실행됩니다.
  • white-space : pre-wrap - 연속 공백은 그대로 유지됩니다. 행의 줄 바꿈은 행의 박스를 채우기 위해 필요한 경우에 실행합니다.

:after

지정한 요소의 뒷 부분에 콘텐츠를 넣을 수 있습니다.

:before

지정한 요소의 앞 부분에 콘텐츠를 넣을 수 있습니다.

box-sizing

박스의 크기를 화면에 표시하는 방식을 변경하는 속성입니다.
  • box-sizing : content-box - 기본값, width와 height 속성은 컨텐츠만 포함하며, padding, border, margin은 포함하지 않습니다.
  • box-sizing : padding-box - width 와 height 속성이 padding 을 포함하고 border, margin 은 포함하지 않습니다.
  • box-sizing : border-box - width와 height 속성이 padding, border 를 포함하고, margin 은 포함하지 않습니다.
  • box-sizing 속성은 스타일 상속이 되지 않습니다.
  • box-sizing은 인터넷 익스플로러, 파이어폭스의 초기 버전에서 23, 크롬에서 height가 window.getComputedStyle()로 계산될 경우와 연관되지 않습니다.

display

inline인지 block인지 등에 관한 해당 요소의 유형을 지정합니다.
  • display : none - 해당 요소가 생성되지 않습니다.
  • display : block - 위아래 줄바꿈이 되는 블럭 요소로 지정합니다.
  • display : inline - 기본값, 위아래 줄바꿈이 되지 않는 인라인 요소로 지정합니다.
  • display : inline-block - 인라인 요소로 배치 되지만 블럭 요소의 특징을 가집니다.
  • display : inline-table - 인라인 요소로 배치 되지만 테이블 요소의 특징을 가집니다.
  • display : list-item - 블럭 요소로 배치되면서 list 항목을 인라인으로 만듭니다.
  • display : run-in - 해당 문맥에 의존하는 블럭 또는 인라인 박스를 만듭니다.
  • display : table - 앞뒤 줄바꿈이 되는 테이블처럼 나타냅니다.
  • display : table-caption - 테이블 캡션처럼 나타냅니다.
  • display : table-cell - 테이블 셀처럼 나타냅니다.
  • display : table-column - 테이블 컬럼처럼 나타냅니다.
  • display : table-column-group - 테이블 컬럼 그룹처럼 나타냅니다.
  • display : table-footer-group - 테이블의 푸터 행 그룹처럼 나타냅니다.
  • display : table-header-group - 테이블의 헤더 행 그룹처럼 나타냅니다.
  • display : table-row - 테이블 행처럼 나타냅니다.
  • display :table-row-group - 테이블 행 그룹처럼 나타냅니다.
